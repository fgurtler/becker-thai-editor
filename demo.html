Vowels

To type non-ascii vowels, use the following:

Hotkey * Replacement
ctrl + a ə
ctrl + o ɔ
ctrl + e ɛ
ctrl + u ʉ
* IE users will need to press ctrl + alt + vowel to trigger the substitution.

Tones

To type the tones, first press the key listed below, then the vowel.

Key Tone Example
` low 'a becomes à
' high 'a becomes á
^ rising ^a becomes â
< falling <a becomes ǎ
